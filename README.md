# Афанасьева Екатерина Витальевна

## Курсовая работа
### «Разработка системы мониторинга эмоционального состояния с интеллектуальными рекомендациями по формированию поведенческих паттернов»

## Реализованные модули

### Авторизация и личный кабинет
- Регистрация и вход пользователя
- Автоматическая авторизация
- Редактирование имени и изображения профиля
- Формирование PDF-отчёта

`AuthViewController`, `ProfileViewController`

---

### Контентный модуль
- Статьи и подкасты
- Голосовое воспроизведение материалов
- Регулировка скорости озвучивания
- Deep linking для Яндекс Музыки

`ExploreViewController`

Технологии:
- `AVFoundation`
- `AVSpeechSynthesizer`
- `UICollectionView`

---

### Психологические тесты
- Последовательное отображение вопросов
- Автоматический подсчёт результатов
- Интерпретация состояния пользователя
- Локальное сохранение результатов

Контроллеры:
- `BeckTestViewController`
- `BeckAnxietyTestViewController`
- `MindfulnessTestViewController`
- `WellBeingTestViewController`

---

### Карта желаний
- Добавление изображений и текста
- Перемещение и масштабирование элементов
- Автоматическое сохранение состояния
- Обновление через `NotificationCenter`

## Дизайн приложения

Интерфейс приложения выполнен в минималистичном бежево-кофейном стиле с использованием мягкой цветовой палитры и декоративного цветочного паттерна на фоне.  
В дизайне применяются скруглённые карточки, полупрозрачные элементы и тёплые акцентные цвета, формирующие спокойную и комфортную визуальную среду для пользователя.

Основные особенности:
- бежево-кофейная цветовая палитра;
- декоративный floral-pattern фон;
- скруглённые карточки и мягкие тени;
- минималистичный интерфейс;
- единая дизайн-система для всех модулей приложения.
