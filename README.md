# Serevia

Система мониторинга эмоционального состояния с интеллектуальными рекомендациями по формированию поведенческих паттернов

Serevia — это мобильное iOS-приложение, предназначенное для поддержки ментального здоровья пользователя, отслеживания эмоционального состояния и формирования полезных поведенческих привычек.

Проект объединяет в единой экосистеме:

- психологические тесты;
- AI-психолога;
- дневник настроения;
- систему аналитики;
- карту желаний;
- медитации и практики осознанности;
- персонализированный контент;
- трекер привычек.

Главная особенность приложения — полностью локальная работа без серверной инфраструктуры. Все пользовательские данные хранятся исключительно на устройстве пользователя, что обеспечивает высокий уровень конфиденциальности.

# Описание

## Возможности приложения

### Авторизация и профиль

- локальная регистрация и авторизация;
- автоматический вход после повторного запуска;
- редактирование профиля;
- генерация аватара по инициалам;
- аналитика эмоционального состояния;
- график настроения MoodChartView;
- экспорт результатов тестов.

### AI-психолог

- локальная AI-классификация эмоционального состояния;
- поддержка русского языка;
- контекстный анализ сообщений;
- персонализированные рекомендации;
- история диалогов;
- работа полностью офлайн.

### Дневник настроения

- создание записей;
- отслеживание эмоциональной динамики;
- хранение истории настроения;
- аналитика пользовательского состояния.

### Психологические тесты

Реализованы:

- BDI — шкала депрессии Бека;
- BAI — шкала тревожности Бека;
- MAAS — тест осознанности;
- PWB — шкала психологического благополучия Риффа.

Каждый тест включает:

- пошаговую навигацию;
- автоматический подсчёт результата;
- интерпретацию состояния;
- сохранение истории тестирований.

### Контентный модуль

- статьи по психологии и самопомощи;
- подкасты;
- озвучивание статей;
- регулировка скорости воспроизведения;
- карточная система контента;
- поддержка длинных материалов.

### Медитации

- библиотека практик;
- таймер медитаций;
- фоновые мелодии;
- управление воспроизведением;
- сохранение состояния сессии.

### Карта желаний

- интерактивный холст;
- drag-and-drop элементы;
- текстовые заметки;
- добавление изображений;
- масштабирование объектов;
- сохранение результата в галерею.

### Трекер привычек

- создание привычек;
- ежедневные отметки выполнения;
- система стриков;
- автоматический сброс статуса;
- мотивационные цитаты.

## Архитектура проекта

В проекте используется архитектурный паттерн MVC.

Структура разделена на:

- Models;
- Views;
- Controllers;
- Services;
- Extension;
- Resources;
- App.

Для взаимодействия между модулями используется NotificationCenter, что позволяет избежать жёстких зависимостей между экранами и сделать архитектуру более гибкой.

Основные уведомления:

- userDataDidChange;
- wishMapDidUpdate.

# Установка

## Для пользователя

1. Склонируйте репозиторий:

```bash
git clone https://github.com/Anastasiien/serevia
```

2. Перейдите в папку проекта:

```bash
cd serevia
```

3. Откройте файл проекта:

```bash
open serevia.xcodeproj
```

4. Запустите приложение через Xcode.

---

## Для разработчика

Если вы хотите участвовать в разработке проекта:

1. Создайте fork репозитория.
2. Создайте отдельную ветку:

```bash
git checkout -b feature/your-feature
```

3. Внесите изменения.
4. Проведите тестирование.
5. Создайте Pull Request.
